# Sistema de Reservas com Google Calendar

## Visão Geral
Sistema de reservas para barbearia com integração ao Google Calendar.
Permite que clientes façam agendamentos com barbeiros disponíveis,
evitando conflitos de horário e sincronizando eventos automaticamente.

## Funcionalidades
- Login com Google (OAuth 2.0)
- Reservas com controle de concorrência
- Integração com Google Calendar
- Bloqueio de horários simultâneos
- Sessão via JWT

## Tecnologias
- Node.js
- TypeScript
- PostgreSQL
- Google Calendar API
- JWT
